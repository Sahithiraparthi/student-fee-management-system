# student-fee-management-system
Java-based Student Fee Management System using Servlets, JDBC, and MySQL for managing student records, fee payments, and admin operations.

## Features

* Admin Login
* Add Student
* Edit Student Details
* Delete Student Records
* View Student Information
* Search Student
* Fee Payment Tracking
* Due Fee Management
* Accountant Management

## Technologies Used

* Core Java
* Servlets
* JDBC
* MySQL
* HTML
* CSS

## Project Structure

* beans → Model/Data classes
* dao → Database operations
* servlets → Request handling and business logic
* WebContent → Frontend pages

## Purpose

This project was developed to simplify fee management operations in educational institutions by providing centralized student and payment management functionality.
